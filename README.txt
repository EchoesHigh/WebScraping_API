This Web Scraping program is specifically built to extract data from the dragonball.fandom JSON API.

Requirements:
Have Node.js installed in your computer.

How to use:
1.- Run "npm install puppeteer" in the terminal (inside this project folder), this should create a folder named "node_modules".
2.- Run "node index" in the terminal (inside this project folder), this should start the web scraping and get you the data in a file named "ScrapedData.js".

If having any problem:
1.- Delete the next files from the project folder:
a) package-lock.json.
b) package.json.
2.- Follow the instructions of "How to use" section in order.
